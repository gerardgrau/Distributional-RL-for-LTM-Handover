# Manuscript

LaTeX source for the paper

> **Risk-Aware Distributional Reinforcement Learning for 5G-Advanced Handover
> Decisions** — Gerard Grau Garcia.

## Contents

| File | Description |
| --- | --- |
| `paper.tex` | IEEEtran (`conference`) manuscript. References are inline (a manual `thebibliography`), so there is no external `.bib` file. |
| `paper.pdf` | Compiled manuscript, built from `paper.tex` (kept in sync with the source). |
| `risk_frontier.png` | Risk–reliability frontier figure. Regenerated by `src/tools/plot_risk_frontier.py`. |
| `master_bar_plots_1col.png` | Final-KPI comparison bar chart. Regenerated by `src/tools/generate_final_plots.py`. |

The two PNGs are the frozen copies used in the submitted PDF. The scripts that
produce them write to `results/final_metrics/plots/`; refresh those copies here
if the figures are regenerated.

## Build

```bash
pdflatex paper.tex && pdflatex paper.tex   # twice, to resolve cross-references
```

Requires a TeX distribution with the standard `IEEEtran` class (TeX Live,
MiKTeX, or Overleaf). No additional packages beyond `cite`, `amsmath`,
`graphicx`, `textcomp`, `xcolor`, and `url`.
